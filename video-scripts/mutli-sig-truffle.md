# setup

```
https://www.trufflesuite.com/docs/truffle/getting-started/installation
npm install -g truffle

mkdir solidity-multi-sig-wallet
cd solidity-multi-sig-wallet

truffle init

npm init -f
npm i --save-dev chai chai-as-promised
```
